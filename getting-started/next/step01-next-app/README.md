This lesson is a [Next.js](https://nextjs.org/) 16 App Router application.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

The root layout is in `src/app/layout.tsx`; the product page is in `src/app/page.tsx`. The page is a client component because its share and notification actions use browser APIs.
