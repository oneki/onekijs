# onekijs
An opinionated framework based on React, React Router, Redux, Redux Saga and Immer.
https://oneki.github.io

## Development

The active workspace targets Node.js 22.22+, React 19, React Router 8, and Next.js 16.

```text
npm ci
npm run typecheck
npm run build
```

See [MIGRATION.md](MIGRATION.md) for the React 19 migration notes and App Router integration requirements.
